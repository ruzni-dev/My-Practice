# MERN-Stack-Development

A learning and practice repository for MERN (MongoDB, Express, React, Node) stack development. This collection contains progressive exercises and small projects focused on HTML, CSS, and front-end foundations as part of a MERN learning path.

## Overview

- **Purpose:** Workspace for learning web fundamentals and practicing tasks that lead into MERN stack development.
- **Scope:** Introductory HTML/CSS lessons, small projects, and structured exercises (front-end first), organized by sections and lessons.

## Contents (high level)

- `2.*` to `7.*` folders — step-by-step HTML and CSS lessons and mini-projects
- `Section-*` folders — grouped resource collections and extras
- `assets` / `public` — static assets used by lessons

Explore the tree in this repository for individual lesson folders (each contains an `index.html` and supporting files where applicable).

## Getting started

1. Clone or open this repository in your editor.
2. Open the lesson folder you want to work on, for example: `2.1-Heading-element/`.
3. Open `index.html` in a browser (double-click or use Live Server extension) to view the lesson.

## Learning goals

- Understand HTML elements and document structure
- Practice CSS selectors, layout, box model, and responsive basics
- Build small projects that demonstrate learned concepts
- Prepare for moving into JavaScript and full MERN-stack topics

## Tasks & exercises

- Complete lessons in numerical order for a progressive learning path
- Copy and modify example files to experiment safely
- Create small variations of the projects (styling, layout, content)
- Add notes or solutions as new files inside each lesson folder

## Contributing / Notes

- This repository is personal learning material. Feel free to adapt it for your studies.
- To add new exercises, create a new folder following the naming convention used by existing lessons and add an `index.html`.
